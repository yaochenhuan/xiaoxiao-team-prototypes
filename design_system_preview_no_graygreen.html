Open Issues

当前仍未确认的问题清单，需在 Figma MVP 搭建前解决。

P0
蓝色 / 紫色完整 0–100 Primitive Tonal Palette 仍未完全补齐
关键 Tone（10/30/40/80/90）已确认，其余 Tone 为 M3_GENERATED 或 PDF 拾取
P0
Success / Warning / Info 色值仍是 TODO_CONFIRM
已建立轻量 Token，但色值未从 Alert 组件或独立色板确认
P1
surface-container / surface-container-high 色值仍需确认
用户已确认保留此系列 Token，但具体色值需从 Figma 源文件导出
P1
Text Field 本体尺寸是否复用 Search 尺寸体系需确认
当前使用搜索框尺寸推断（40/48/56），Text Field 本体未在 PDF 中直接标注
P1
Card 精确尺寸需从源设计稿确认
当前采用 M3 标准作为临时建议（Radius 12, Padding 16），源设计稿无明确标注
P1
Dark Mode 当前不做，作为 v1.5 / v2.0 规划
v1.0 阶段仅做 Light Theme（蓝/紫 2 套），Dark Mode 无设计稿
